# Text Encryption & Decryption Tool 🔐

A lightweight Python application designed to automate text encryption and decryption processes. This project showcases my understanding of data manipulation, string operations, and fundamental algorithmic logic in Python.

---

### 🚀 Features
- **Automated Encryption:** Quickly transforms plain text into secure ciphertext.
- **Automated Decryption:** Reverses the process to retrieve the original text instantly.
- **User-Friendly CLI:** Simple and interactive command-line interface.

---

### 🛠️ Tech Stack
- **Language:** Python 3.x

---

### ⚙️ How to Run

Copy and run the following commands in your terminal to clone and run the project:

```bash
git clone https://github.com/saifaliobayes/Encryption-Decryption.git
cd Encryption-Decryption
python Encryption.py
```
